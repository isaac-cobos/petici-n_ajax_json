Esta es la prueba técnica que hemos diseñado para que demuestres tus conocimientos de JavaScript, jQuery & AJAX

Para realizar la prueba de manera satisfactoria deberás...

  • Convertir, a través de una petición AJAX, el documento HTML estático que te entregamos en una app front consumiendo los datos dinámicamente de la carpeta data a través de una petición AJAX.
  • Todos los archivos externos que necesitas están incluidos en la prueba, pero deberás enlazarlos en la parte del documento que mejor consideres.
  • Deberás obviar los datos que no sean necesarios.
  • Deberás dar el mismo formato (mayúsculas/minúsculas) a los datos que el ejemplo estático.
  • Deberás mostrar el precio proveniente del JSON con un 21% de IVA ya incluido.
  • El listado solo deberá aparecer en el DOM cuando pulse el botón de "Mostrar Listado"
